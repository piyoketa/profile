説明文長文
東大の卒業論文でインタビュー分析手法（現象学的質的分析）を学んで以降、趣味でインタビュー分析を続けています。「n=1をできる限り高い解像度で描く」を標語に、本人の思考の流れや価値観を丁寧にたどり、本人でも言語化できていない部分を描写することを目指しています。
